🏧 ATM Incident MIS Report — May–June 2026

A Management Information System (MIS) report tracking ATM incidents, downtime, SLA compliance, and agent performance across 6 major cities in India.

📊 Project Overview

This report was built to provide operational visibility into ATM network health — helping stakeholders identify recurring fault patterns, monitor SLA adherence, and evaluate field agent performance.

Metric Value Total Incidents Tracked 100, ATMs Monitored 20, Cities Covered 6 (Mumbai, Delhi, Chennai, Bangalore, Kolkata, Hyderabad)Reporting Period May 1 – June 6, 2026 Average Downtime 98.1 Minutes Maximum Downtime 180 Minutes
